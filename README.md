# Majestic Bytes

A premium technical assistant for Cheat Engine automation and memory manipulation. Built with Tauri, React, and TypeScript.

## Features

- **Hex Converter**: Bidirectional Hexadecimal and Decimal conversion.
- **AOB Signature Generator**: Compare memory arrays and generate signatures with automatic wildcards.
- **Lua Script Generator**: Generate Cheat Engine compatible Lua scripts with optimized scanning templates.

## Development

To run the application in development mode:

```bash
npm run dev
```

## Build and Package

To build the production executable:

```bash
npm run pack
```

The generated executable will be located in the `src-tauri/target/release/bundle` directory.
